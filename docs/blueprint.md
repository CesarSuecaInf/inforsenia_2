# **App Name**: INFORSENIA Fest

## Core Features:

- Event Homepage: A prominent landing page displaying the event title, dates (May 13-14), the core theme, and a clear call to action.
- Comprehensive Schedule: A dedicated section presenting the detailed agenda for Day 1 (talks/presentations) and Day 2 (workshops), including session times, titles, brief descriptions, and associated speaker information.
- Speaker & Presenter Directory: Individual profiles for each speaker and workshop leader, featuring their photo, biography, and a list of their respective sessions.
- Venue & Logistics Information: Detailed practical information about the event's location (IES La Sénia in Paiporta, Valencia), travel instructions, and other essential attendee logistics.
- Sponsors & Collaborators Showcase: A section to proudly display the logos and links of partners and sponsors who are supporting the INFORSENIA Fest.
- AI Event Assistant: An interactive chatbot tool, powered by generative AI, designed to answer common attendee questions regarding the schedule, speakers, and general event logistics by utilizing the website's content.

## Style Guidelines:

- Scheme: Dark. This choice emphasizes a professional and modern technological aesthetic.
- Primary color: Deep, innovative violet (#5544F9). Selected to represent depth of knowledge and cutting-edge innovation.
- Background color: A subtly violet-tinged dark grey (#1E1C2E). This provides a sophisticated and non-distracting backdrop for content.
- Accent color: Bright, vivid sky blue (#59BFFB). Used to highlight interactive elements, calls to action, and important information for contrast.
- Headlines: 'Space Grotesk' (sans-serif) for a modern, tech-inspired, and impactful appearance that draws attention.
- Body text: 'Inter' (sans-serif) for its neutrality, excellent readability, and contemporary aesthetic, suitable for presenting detailed information.
- Code snippets: 'Source Code Pro' (monospace) for clear and legible presentation of any programming-related content, aligning with the technical nature of the event.
- Minimalist and modern outline icons should be used to convey information clearly and efficiently without visual clutter.
- A clean, grid-based layout ensuring excellent readability and a strong visual hierarchy across all device sizes, optimizing for content consumption.
- Subtle, fluid animations for navigation transitions and interactive elements to enhance the user experience without causing distraction.
